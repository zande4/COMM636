This website will show you an example page where CSS grid excels (and behaves as expected with screen size changes) and one page where CSS flexbox is advantageous. You can switch your view of each page to use the other styling method to acheive the closest approximation of the orginal page. You may also toggle box model visualizations.

Box-model information: Each element in the DOM, is rendered with some pixel value for each of the following:

  • Content
  • Padding
  • Border
  • Margin

The only value that can't be directly manipulated is content - the space taken up by what's inside an element. The values are rendered in the order shown above. Border-color styles the border pixels, while background color styles both the content and padding pixels. Margins add space between elements.
